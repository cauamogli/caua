# Portfólio | Cauã Brito

Este é o meu portfólio pessoal desenvolvido para apresentar meus projetos, experiências, habilidades técnicas e trajetória como desenvolvedor.

O objetivo deste projeto é reunir, em um só lugar, informações sobre minha atuação na área de tecnologia, meus principais trabalhos acadêmicos e profissionais, além de demonstrar minhas competências em desenvolvimento web.

## Link do projeto

Acesse o portfólio publicado pelo GitHub Pages:

https://cauamogli.github.io/caua/

## Sobre mim

Meu nome é Cauã Brito, sou estudante de Ciência da Computação e desenvolvedor com experiência em projetos web, aplicações mobile e sistemas voltados para solução de problemas reais.

Tenho interesse em desenvolvimento full stack, criação de interfaces modernas, automações, integração com APIs e construção de sistemas que facilitem processos do dia a dia.

## Tecnologias utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Shadcn/UI
- Git
- GitHub Pages

## Funcionalidades

- Página inicial de apresentação
- Seção sobre mim
- Listagem de projetos
- Página de portfólio
- Layout responsivo
- Interface moderna e animada
- Navegação entre páginas
- Publicação via GitHub Pages
